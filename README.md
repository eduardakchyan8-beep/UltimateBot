# 🚀 UltimateBot — Discord Economy & Leveling Bot

Мощный Discord-бот с системой экономики, уровней и инвентарем.

## ✨ Особенности
- 💰 **Экономика**: Зарабатывайте монеты, работайте и покупайте предметы.
- 📈 **Уровни**: Получайте XP за сообщения и повышайте свой уровень.
- 🎒 **Инвентарь**: Покупайте машины, дома и другие предметы в магазине.
- 🎁 **Промокоды**: Получайте бонусы за секретные коды.
- 🔔 **Напоминания**: Бот напомнит вам в Личные Сообщения или в чат, когда можно снова поработать.
- 🛡️ **Админ-инструменты**: Команды для управления балансом, опытом и инвентарем.

## 🛠️ Команды

### Игровые
- `/rank` — Посмотреть свой профиль, XP и инвентарь.
- `/balance` — Проверить сколько у тебя денег.
- `/work` — Поработать и получить монеты (раз в 30 мин).
- `/daily` — Забрать ежедневный бонус (1,000 монет).
- `/weekly` — Забрать еженедельный бонус (3,000 монет).
- `/shop` — Посмотреть список товаров в магазине.
- `/buy [id]` — Купить предмет из магазина.
- `/promo [код]` — Использовать секретный код.

### Администраторские (Только для владельца)
- `/additem [@игрок] [предмет] [цена] [редкость]` — Выдать предмет игроку.
- `/delitem [@игрок] [id]` — Удалить предмет из инвентаря игрока по номеру.
- `/clear [1-100]` — Быстро очистить сообщения в канале.
- `/setxp [@игрок] [число]` — Установить количество опыта.

## 🚀 Как запустить

1. **Создайте файл `.env`** и добавьте туда:
   ```env
   TOKEN=ВАШ_ТОКЕН_БОТА
   GUILD_ID=ID_ВАШЕГО_СЕРВЕРА
   ```
2. **Установите зависимости**:
   ```bash
   npm install
   ```
3. **Запустите бота**:
   ```bash
   node index.js
   ```
